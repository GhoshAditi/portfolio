# Portfolio Website

A modern, responsive portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- ✨ Modern and sleek design with beige color theme
- 🎭 Smooth animations with Framer Motion
- 📱 Fully responsive design
- ⚡ Fast performance with Next.js
- 🎨 Beautiful UI components
- 📧 Contact form with validation
- 🔗 Social media integration
- 💼 Project showcase with live links
- 📚 Education and experience sections
- 🛠️ Tech stack visualization
- 📖 LinkedIn posts integration

## Sections

1. **Header** - Navigation with smooth scroll
2. **Hero** - Introduction with profile picture placeholder
3. **About Me** - Personal information and education
4. **Tech Stack** - Skills and technologies with progress bars
5. **Projects** - Portfolio projects with live links and GitHub repos
6. **Experience** - Work history and achievements
7. **Services** - What I offer as a developer
8. **LinkedIn Posts** - Recent professional updates
9. **Socials** - All social media platforms and coding profiles
10. **Contact** - Contact form and information

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd portfolio
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

### Personal Information

1. **Replace placeholder content** in all components with your actual information:
   - Update "Your Name" throughout the codebase
   - Add your actual email, phone, location
   - Update social media URLs
   - Replace project information with your projects

2. **Add your profile picture**:
   - Replace the placeholder in `components/Hero.tsx`
   - Add your image to the `public` folder
   - Update the image source

3. **Update projects**:
   - Replace project data in `components/Projects.tsx`
   - Add actual project screenshots
   - Update live URLs and GitHub repositories

4. **Customize colors**:
   - Modify the beige theme in `tailwind.config.js`
   - Update color variables in `globals.css`

### Content Updates

- **About section**: Update your story, education, and stats
- **Tech Stack**: Add/remove technologies and adjust skill levels
- **Experience**: Replace with your actual work experience
- **Services**: Customize the services you offer
- **LinkedIn Posts**: Replace with your actual LinkedIn content
- **Contact**: Update contact information and form handling
